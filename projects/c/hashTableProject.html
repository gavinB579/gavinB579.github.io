C Hash Table Showcase - Inventory and Sales Simulation

This project was a simulation of a an inventory and sales system. It allows for the user to buy, sell, and change the price of items in the inventory. This was done to showcase the use of hash tables and chain nodes. We used a custom hash function to hash, and all data was accessed through the hash values. This program takes in the number of commands, the type of command (BUY, SELL, or CHANGE), the name of the item, and the price of the item (if applicable). The program then manipulates the inventory based on the commands.

This project taught me a lot about hashing and hash tables. It also used a lot of concepts from my previous projects, similar to the other projects I have listed on this website. It included linked lists, dynamic memory allocation, and structs. I had to learn how to use all of these concepts together to create a functioning hash table with chaining.
